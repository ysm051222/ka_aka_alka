# ka_aka_alka

This project used the Korean Forced Aligner as the alignment tool.  
See the [website](https://tutorial.tyoon.net/) for details.

Participants were given an MP3 file (kkak/짝꿍단어로익히는중국어.mp3 or kkak/짝꿍단어로익히는일본어VOCA.mp3) and asked to follow the instructions below.

## Instructions given to participants

안녕하세요?

지시사항은 간단합니다.  
먼저 나이와 출신 지역을 말씀해 주세요. 예: 20대, 경기도

그다음 제시되는 음성 파일을 듣고, 각 항목이 무엇에 해당하는지 답해 주시면 됩니다.

1. 가 vs 카 vs 까
2. 아가 vs 아카 vs 아까
3. 알가 vs 알카 vs 알까

소정의 상품을 드리지 못하는 점 양해 부탁드립니다.
